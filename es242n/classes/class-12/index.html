In-class problems

Work these on your own machine. The take-home practice list (LeetCode) follows below.

1

Subsets, both ways

Write subsets() recursively (include/exclude). Then write the iterative version. Test both on {1, 2, 3, 4}.

2

Try iterative parens

Try to enumerate well-formed parens for n = 2 using the bottom-up insertion idea. Where does it break? Note the first invalid output.

3

Generate Parens in C

Write gen(open, close, n) end-to-end. Test on n = 3; verify the 5 outputs match the Catalan count.

4

Trace and break

Trace gen(0, 0, 3) by hand and draw the call tree. Then break it: comment out the close < open guard and observe the invalid output.
