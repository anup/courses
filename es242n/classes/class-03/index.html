15 May 2026: C Fundamentals and Arrays

Goal for the session: use C syntax, loops, and arrays to retry the diagnostic coding problems from the reopened contests.

01

C fundamentals

Compilation model, variables, input/output with scanf and printf, arithmetic operators, conditionals, and loops.

02

First hands-on checkpoint

Compile a C program, read input, print output, and solve a small loop-based problem before moving into arrays.

03

Arrays

Declaration, indexing, iteration, array size, counting, and simple scans over one-dimensional arrays.

04

Lab

Work on the reopened diagnostic coding contests. Start with UNIQUE-ELEMENT, then move to ARRAY-OPERATIONS. SADDLE-POINT, NAME-CLEAN, and ALTERNATING-PARITY are stretch problems.

05

Wrap

Review common mistakes, discuss one stronger solution on the projector, and set the weekend LeetCode practice.
