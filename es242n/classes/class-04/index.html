Optional Local C Drills

These are not online-judge problems. Use them only to check whether you can write the pointer code locally with gcc.

A

Swap using pointers

Write void swap(int *a, int *b). In main, read two integers, call swap(&x, &y), and print the swapped values.

B

Minimum and maximum through pointers

Write void min_max(int arr[], int n, int *mn, int *mx). Read n, then n integers, and print the minimum and maximum returned through pointer arguments.

C

Return a new reversed array

Write a function that uses malloc to create a reversed copy of an input array and returns the pointer. Print the returned array in main, then call free.
