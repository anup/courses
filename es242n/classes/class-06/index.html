Practice After This Day

Use these problems to practice the linked-list ideas from the slides. Start with the local C exercises, then move to LeetCode after the pointer-based version is comfortable.

A

From the array-based linked-list slides

  • Given data[], nxt[], and head, print the list in order.
  • Insert a value at the front using data[], nxt[], head, and avail.
  • Delete the first occurrence of a value and return the freed slot to the availability list.
  • Write a function that counts the number of elements reachable from head.
B

From the pointer-based linked-list slides

  • Write createNode, traverse, and insertFront for struct Node.
  • Write insertEnd and test it on an empty list and a non-empty list.
  • Write deleteNode for the first occurrence of a value.
  • Write freeList and explain why the next pointer must be saved before calling free.
