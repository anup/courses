29 May 2026: Stacks and Queues — Applications

We finish stacks and queues by putting them to work: postfix evaluation with a stack, and grid traversal with DFS (stack) and BFS (queue). Recursion — which is DFS using the hidden call stack — comes next class, on Monday.

Slides

  • Stacks and Queues in Action — postfix, DFS/BFS on a grid, shortest path
S1

Session 1 — Postfix and the DFS/BFS idea

Postfix (RPN) evaluation with a stack. Then the grid-traversal skeleton: the same code with a stack for "go deep" (DFS) and a queue for "go wide" (BFS).

S2

Session 2 — Grid traversal problems

Flood fill and number of islands, solved both ways. Shortest path if time permits — where BFS earns its keep. Plus time on the practice lists below.

In-class problems

Work these during class on judge.csbasics.in or LeetCode. Start with postfix; the grid problems build on the same traversal idea.

1

Evaluate postfix (RPN)

LeetCode 150. Push operands; on an operator, pop two, apply, push the result. The first value popped is the right operand — watch the order for − and /.

2

Flood fill

LeetCode 733. Recolour a connected region. Either DFS (stack) or BFS (queue) works — you only need to reach every cell.

3

Number of islands

LeetCode 200. Count connected groups of land. Solve it once with a stack and once with a queue — notice the code barely changes.

4

Shortest path in a grid (if time)

LeetCode 1091. This is where BFS earns its keep: explore in waves and the first arrival at the target is the shortest path. DFS does not give this.

Practice — stacks: simulate with push and pop

Scan the input once; push and pop to track state. The "second problem" family — more of the same idea.

  • 682. Baseball Game (Easy) — apply a record of operations to a stack of scores.
  • 1047. Remove All Adjacent Duplicates In String (Easy) — pop when the top matches the current character.
  • 844. Backspace String Compare (Easy) — # means backspace; build each string on a stack, then compare.
  • 1544. Make The String Great (Easy) — pop adjacent pairs that differ only in case.
  • 2390. Removing Stars From a String (Medium) — * pops the most recent character.

Practice — stacks: matching and parentheses

  • 20. Valid Parentheses (Easy) — the canonical matching problem.
  • 1021. Remove Outermost Parentheses (Easy) — track depth with a counter or stack.
  • 921. Minimum Add to Make Parentheses Valid (Medium) — count unmatched brackets.
  • 1249. Minimum Remove to Make Valid Parentheses (Medium) — stack of indices to delete.

Practice — grid traversal (DFS and BFS)

  • 695. Max Area of Island (Medium) — like number of islands, but track the size of each region.
  • 1020. Number of Enclaves (Medium) — land cells that cannot reach the boundary.
  • 130. Surrounded Regions (Medium) — flip regions not connected to the border.
  • 994. Rotting Oranges (Medium) — multi-source BFS; all rotten cells spread in waves at once.

Practice — queues

  • 933. Number of Recent Calls (Easy) — a queue as a sliding time window.
  • 622. Design Circular Queue (Medium) — extends today's array-backed circular buffer.
  • 649. Dota2 Senate (Medium) — two queues taking turns.
  • 950. Reveal Cards In Increasing Order (Medium) — simulate the reveal with a queue of indices.
