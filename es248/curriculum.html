How multi-byte values are stored in memory (little-endian vs big-endian). Directly relevant to understanding how C reads bytes when you cast pointer types, which was demonstrated in the Pointers slides.

Mid-Semester Exam

The mid-semester examination will be conducted entirely online.

Part A — Multiple Choice Questions

Conceptual questions covering Linux basics, C fundamentals, memory, data types, pointers, arrays, and strings.

Part B — Coding Questions

Problems to be solved on an online coding platform. Focus on writing correct C programs involving pointers, arrays, and strings.
